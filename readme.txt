TheChecker.co integration for Contact Form 7

== Description ==
This plugin validate email in Contact From 7 

[TheChecker.co](https://thechecker.co/?coupon=JAWOROWICZ)
Remove spam traps, fake, invalid, old and unused email addresses using our website or API and thus prevent bad reputation penalties from your email service provider.

Officially approved integration.

Use: 
1) Enter the Public API Code to Themes > TheChecker.co CF7
2) Add or edit existing Contact form and put the shortcode
\'[email* email class:thechecker-input]\'
or add a \'class:thechecker-input\' to existing email shortcode 
or Use a shortcode generator \'emailchecker\'
3) Save Form 
4) Preview and done!

For Expert Uses:
You can fully customize your widget and validiations error just setting some params in javascript. 

Full instruction and review (Lang PL):
[http://jaworowi.cz/thechecker-co-wordpress-10432.php](http://jaworowi.cz/thechecker-co-wordpress-10432.php)


Free 1000 credit / 1000 email check for new users:
Use JAWOROWICZ coupon code in signup form or [this link](https://thechecker.co/?coupon=JAWOROWICZ).

The plugin and integration work fully with SalesManago CF7 Integration.

== Installation ==
1. Upload \"thechecker-cf7.zip\" to the \"/wp-content/plugins/\" directory.
1. Confirm that [Contact Form 7](https://wordpress.org/plugins/contact-form-7/) is installed and activated. Then activate this plugin.
1. Edit a form in Contact Form 7.
1.Add or edit existing Contact form and put the shortcode
\'[email* email class:thechecker-input]\'
or add a \'class:thechecker-input\' to existing email shortcode 
or Use a shortcode generator \'emailchecker\' and Insert the generated tag anywhere in your form.
